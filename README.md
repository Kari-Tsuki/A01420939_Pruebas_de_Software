# A01420939_Pruebas_De_Software
# Actividad A4.2 – Ejercicios de Programación

**Lenguaje utilizado:** Python 3.14.3

## Contenido y evidencias

Este repositorio contiene los tres ejercicios de programación solicitados para la actividad A4.2. Para cada ejercicio se incluye:

- **Código fuente en Python** (*.py*).
- **Capturas de terminal** mostrando la ejecución de cada caso de prueba (TC), dentro de la carpeta `results`.
- **Capturas del análisis con PyLint** de cada ejercicio, también en `results`.
- **Archivos de resultados de cada caso de prueba** (*.txt*), renombrados manualmente para identificar cada TC, por ejemplo: `Resultados_TC1.txt`.

## Observaciones

- Todos los programas cumplen con el estándar de codificación PEP-8 y pasaron PyLint, con excepción de un detalle menor de nombres que se ignoró por requerimiento de la actividad.
- Los archivos de resultados `.txt` fueron renombrados manualmente para cada TC, documentando claramente los resultados individuales.
- Toda la evidencia presentada permite verificar la correcta ejecución y el cumplimiento de las especificaciones de la actividad.
